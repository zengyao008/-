# 商用车融资租赁资产监控AI助手
面向融资租赁授信评审场景的一站式资产质量分析工具，基于Python+Streamlit+豆包大模型构建，实现台账数据自动清洗、风险指标一键测算、异常智能预警、专业报告自动生成全流程闭环。

## 项目背景
针对商用车融资租赁业务资产监控中，手工计算效率低、统计口径不统一、风险识别滞后、分析报告产出慢等痛点，打造标准化、自动化的分析工具，赋能风控日常工作。

## 核心功能
### 1. 数据智能清洗
- 支持租金收入表、资产余额表双表自动关联匹配
- 内置数据校验规则，自动识别异常数据
- 支持多维度业务筛选（业务类别、业务模式、大区等）

### 2. 多维度风控测算
- **Vintage账龄分析**：按放款批次追踪逾期演化轨迹，严格遵循同账龄横向对标原则
- **迁徙率分析**：月度逾期档位迁徙矩阵，量化风险传导速度与路径
- **损失率与拨备测算**：基于迁徙率模型计算各级损失率，自动计提月度拨备

### 3. 风险智能预警
- 持续恶化账户识别：精准定位逐期逾期升级、无还款回落的高风险资产
- 大额逾期Top榜：按未偿还本金排序，聚焦核心风险敞口
- 异常波动自动标记，辅助风控决策

### 4. AI智能解读
- 基于豆包大模型，自动生成专业级月度资产质量分析报告
- 严格遵循风控统计规则，所有结论均有数据支撑
- 支持一键导出报告文本，直接复用至工作汇报

## 技术栈
- 前端交互：Streamlit
- 数据处理：Pandas
- 可视化：Plotly
- 大模型：火山引擎方舟 · 豆包大模型
- 部署：支持本地部署 / Streamlit Cloud在线部署
